Gupshup

WhatsApp clone web app where any person can login or signup using their email and can chat with any person who is already registered and can share media such as images, videos, document along with text messages.It is using React for frontend and Context API for managing various states, firebase firestore as db for storing information related to user and their chats, firebase cloud storage for storing profile pictures, images, documents and other media and using firebase hosting for hosting this web app.
